Ubuntu 删除用户

装了Ubuntu系统还不熟悉，右上角的一个下三角下面，我账号下面的账号设置菜单里，解锁，把我的用户名后面的2019删掉了，确认，….尼玛多了个用户



这我有点强迫症啊，非得要删掉，在此感谢下面的博主
https://blog.csdn.net/qq_33867131/article/details/86490083 https://blog.csdn.net/CAU_Ayao/article/details/83502880
我多的用户是gxq，故要删除它
首先我的Ubuntu系统是18.04版本的desktop，桌面右键，打开终端，执行命令

sudo userdel gxq

然后删除/home下的gxq文件夹

sudo rm -f /home/gxq

然后博客上说，删除用户权限相关配置，即/etc/sudoers中的关于gxq用户的配置，否则无法在此创建同名用户
说实话我也看这个文件下的配置了，实在没看出来有啥单独关于gxq用户的配置，基本就是赋予普通用户的权限的，所以我没做此步骤，建议有的话删掉
接着，删除gxg用户的残留信息
首先转到home目录下

cd /home

查看/etc/passwd的关于gxq的配置，删掉

cat /etc/passwd

然后删除/etc/group下的关于gxq的配置，在这遇到了一个小小的坑，就是蓝色部分字体我也删了。。。。。。。类似于下图


最后删除/var/spool/mail下的邮箱文件

cd /var/spool/mail

rm -rf gxq

Ok,Over!!!!!!!!!!
