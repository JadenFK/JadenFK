Ubunt18.0.4安装Mysql及Navicat

Ubuntu安装MySql 5.7.26
首先感谢百度经验：https://jingyan.baidu.com/article/6c67b1d64dbd7d2786bb1e6d.html 和
首先我以为和Windows一样呢，在某个文件夹下安装MySQL就会安装在那，事实上我错了，我实在/home/gxq2019/software/mysql文件夹下打开的终端，然后转到的root用户，结果安装还是自动

正式开始

1 右键打开客户端，运行下面的命令转到用户root下，其间需要输入root用户的密码

su root


2 运行代码,进行更新命令

sudo apt update

3 更新完成后，查看一下MySql的服务状态

sudo service mysql status

4 查看mysql版本号、设置密码和远程登录

sudo mysql //查看MySQL版本号

set password for ‘root‘@’localhost’ = password(‘root’) //在mysql>下输入设置密码

GRANT ALL PRIVILEGES ON . TO ‘root‘@’%’ IDENTIFIED BY ‘root’ WITH GRANT OPTION //此root是上面的密码，设置允许远程登录


5 更改MySQL的配置文件

在更改文件之前，先退出上面的mysql>命令，输入exit;按enter键即可退出
然后关闭MySQL服务，执行下面的命令

sudo service mysql stop

接着，执行
sudo vim /etc/mysql/mysql.conf.d/mysqld.cnf //事先安装好vim

就进入了vim编辑器，i:插入；esc:退出插入；:w是保存，:wq是保存退出；:q是退出
注释掉bind-address = 127.0.0.1即可，保存退出


#### 6 启动MySQL服务
sudo service mysql start

7 远程测试

IP地址在Ubuntu中查看即可，如下：
如果运行ifconfig找不到此命令，则没有net-tools工具，先下载，

sudo apt-get install net-tools
